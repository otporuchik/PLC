тестовое приложение

Подключение модуля дискретных выходов 32 (МУ110-220.32Р):
32бит аутпут модуль
регистр: 0
команда: Форс мультиплай коилс (0х0f)
использование в коде: присвоенное имя модуля.(точка)номер бита от 0 до 31.

Подключение модуля дисркетных вводов 32 (МВ110-220.32ДН):
32бит инпут модуль
регистр: 99
команда: рид инпут реджистерс (0х04)
использование в коде: присвоенное имя модуля.(точка)номер бита от 0 до 31.

Подключение модуля дискретных вводов 16 (МВ110-224.16ДН):
реджистер инпут модуль
регистр: 51
команда: рид инпут реджистерс (0х04)
использование в коде: присвоенное имя модуля.(точка)номер бита от 0 до 15.

Подключение модуля дискретных выводов 16 (МУ110-224.16Р):
реджистер оутпут модуль
регистр: 50
команда: райт мультиплай реджистерс (0х10)
использование в коде: присвоенное имя модуля.(точка)номер бита от 0 до 15.

Подключение модуля аналоговых выходов 8 (МУ110-224.8И):
реджистер аутпут модуль (для каждого выхода создается отдельный)
регистр: для каждого модуля свой - от 0 до 7 (соответств.выходы: 0...8)
команда: 0х10
использование в коде: по имени реджистер аутпут модуля с соответствующим регистром.
переменная типа ворд, значение 0...1000 соответствует выходу 4...20мА.

ПЛК160:

Работа с аналоговыми выходами на ПЛК160.
Переменные типа РЕАЛ, значение от 4 до 20.
Схема подключения для ПЛК типа А.М. - универсальная, не как для И (где только токовый выход).
