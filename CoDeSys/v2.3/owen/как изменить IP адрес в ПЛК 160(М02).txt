1. Подключись к ПЛК
2. В меню "Онлайн" выбери "Читать файл из ПЛК"
3. В открывшемся меню выбери место, где будет сохранен скачиваемый файл (при желании), это
верхнее меню с выпадающим списком.
4. В окне "Имя файла" введи: local_addres.dat
Внимательно следи за написанием, если ошибешься, эта падла создаст файл с ошибочным именем
и все, ничего путного при этом не выйдет!
5. Открой скачанный файл.
Его содержимое по умолчанию:

EMAC=e4:1e:0a:00:e4:7b
IP=0A:00:06:0A
GATE=0A:00:06:01
MASK=FF:FF:FF:00
DNS=8:8:8:8
DNS=8:8:4:4

6. В строке IP замени адрес на требуемый
7. Сохрани файл.
8. В меню "Онлайн" выбери "Записать файл в ПЛК".
9. Выбери измененный и сохраненный файл.
10. Отключи и включи питание ПЛК.
11. Готово, у него новый IP адрес.
